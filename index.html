Mobile Weather App

Mobile Development

Mobile Weather App

Description:

The objective was to create a mobile application to consult the weather in Portugal. In this project, the IPMA API was used to obtain meteorological data and these data are displayed in the app. The App makes requests to the API once a day when the district is selected and saves that information to show later if the user selects the district again. It is possible to set a district as favorite and these districts are placed at the top of the list of districts. It is also possible to share the weather status of a given day with other people. This app is only for personal use.

Main Activities:

Analysis and architecture planning;

Mobile Development (Android);

Technologies:

Java; SQLite
